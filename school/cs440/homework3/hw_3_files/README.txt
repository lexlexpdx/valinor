Instructions for creating virtual environment for this assignment

1. Run the following command:

    ./python_env.sh

    This will set up all required libraries for this assignment
2. Run the following command:

    source myenv/bin/activate

    This activates the virtual environment


Note:
- All figures required by the assignment that were not included in the jupyter notebook/PDF
are located in the "figures" directory.
- I have also included a PDF of the Jupyter Notebook if, for some reason the file will not load.
